# 🚗 Car Management System

C# da yozilgan oddiy avtomobil boshqaruv tizimi. OOP tamoyillarini amalda qo‘llash uchun yaratilgan.

---

## 📋 Loyiha haqida

Ushbu loyiha quyidagi mavzularni o‘z ichiga oladi:
- class  
- object
- field 
- method
- namespace 
- using

---

## 🚀 Funksiyalar

| Metod | Vazifasi |
|-------|----------|
| `ShowInfo()` | Mashina haqida to‘liq ma'lumot chiqaradi |
| `Drive(distance)` | Mashinani yurgizadi (10 km = 1% yoqilg‘i) |
| `Refuel(amount)` | Yoqilg‘i quyadi (100% dan oshmaydi) |
